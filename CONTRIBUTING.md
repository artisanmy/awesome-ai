# Contributing to Awesome Artisan AI

Thanks for your interest in contributing! This is a curated list of resources for Artisan AI. We keep quality high and spam out.

## Guidelines

### Before You Submit

- **One item per PR** — Each pull request should add exactly one item to the list.
- **Genuinely useful** — The resource must be notable and valuable to the community. No self-promotion, spam, or low-quality content.
- **Link must be HTTPS** — All URLs must use HTTPS. Dead links will be rejected.

### Formatting

Each entry follows this format:

```
- [Name](https://url) - Single sentence description, starts capital, ends period.
```

**Description requirements:**
- Single sentence only
- Starts with a capital letter
- Ends with a period
- No marketing language ("world-class," "revolutionary," "amazing," etc.)
- Describe what the resource is or does, not why it's great

**Example:**
- [Artisan Docs](https://artisan.com) - Comprehensive guide to Artisan AI platform features and API endpoints.

### Organization

- Add your item to the correct category in **alphabetical order** by name.
- New categories require a minimum of 5 items to be created.

### Submitting

1. Fork the repository
2. Create a branch (`git checkout -b add/resource-name`)
3. Add your entry following the format above
4. Commit with a clear message
5. Open a pull request

We'll review and merge if everything checks out!

## Questions?

Open an issue or reach out. We're here to help.
